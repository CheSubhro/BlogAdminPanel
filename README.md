# Blog Admin Panel (Full-Stack)

A powerful and modular admin dashboard for managing blog content, built with the MERN stack and optimized for performance.

## 🚀 Tech Stack

- **Frontend:** React (Vite)
- **Backend:** Node.js & Express.js
- **Database:** MongoDB
- **API Testing:** Bruno

## ✨ Key Features

- **Full CRUD:** Create, Read, Update, and Delete blog posts.
- **Vite Setup:** Fast HMR and optimized build process.
- **Organized API:** All endpoints are categorized and tested using Bruno collections.
- **Clean Architecture:** Folder-wise separation for Backend and Frontend.

## 📁 Project Structure

- `/frontend`: React application with Vite.
- `/backend`: Node.js server and API routes.
- `/api-collection`: API testing collections.

## 🛠️ Installation

1. Clone the repo: `git clone [your-repo-link]`
2. Install dependencies: `npm install` (both in root/frontend/backend)
3. Set up your `.env` for MongoDB.
4. Run the development server: `npm run dev`
